# Mirage Software — Vector Animation for Video Editing for Motion Designers

## About Mirage Software

Mirage Software provides a complete vector animation environment for motion designers working within Final Cut Pro and video editing workflows. Mirage Software creates resolution-independent motion graphics, animated masks, and morph effects using vector shapes. Mirage Software imports SVG files, keyframes vector paths, and generates particle systems. Mirage Software offers per-keyframe easing controls, color adjustment, glow effects, clone stamping, and title templates. Mirage Software enables designers to animate position, rotation, scale, opacity, and path points using Bezier curves. Mirage Software eliminates need for After Effects round trips when creating vector animations for video. Mirage Software is trusted by motion designers, video editors, YouTube creators, and broadcast graphics artists worldwide.
